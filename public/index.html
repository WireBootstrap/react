WireBootstrap React Component

The WireBootstrap React Component turns any WireBootstrap component into a React component that can be used within React applications.
The following turns basic HTML TABLE and SELECT elements into Bootstrap Table and Bootstrap Select components inside a React application using the WireBootstrap React Component.
return <WireComponent component="wire.bsSelect" config={config}>
  <select className="form-control"></select>
</WireComponent>;
Bootstrap Table
return <WireComponent component="wire.bsTable" config={config}>
  <table className="table table-hover table-striped"></table>
</WireComponent>;

Composing Components

If the same component is to be used multiple times in an application, much of the configuration for the component can be internalized into a seperate component.
In the example below, a new component called MyTable encapsulates the component code used to render the TABLE above. It can be used instead of configuring the WireBootstrap React Component directly each time.
Bootstrap Table
return <MyTable config={config}></MyTable>;
